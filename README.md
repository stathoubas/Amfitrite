# Amfitrite tools
Developed AI tools & analyses for the Amfitrite project

## Contributing Guide

### Branching Workflow

- Do all work in the `dev` branch: push, commit, experiment.
- When a milestone is ready:
  - Clean your notebooks and code
  - Submit a pull request from `dev` to `main`
  - At least one teammate must review the PR
  - Optionally, update README with changes
